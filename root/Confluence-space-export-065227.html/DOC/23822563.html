Documentation : Case 1: Lot Wise (Include Color = Yes)

Created by Siddharth Rastogi, last modified on Mar 24, 2018

Data Points & Logic

S.NoData Points Logic
1No. of Pairs 

The dropdown value will give the list of numbers in multiple of Lot Size Value. 

Minimum being the Lot size value itself and going upto 10 multiples.

  • Validate for Stock Lot information. If Stock lot is activated and the value for a particular size set is say 3 ; where lot size = 8, then in the dropdown, the values populated should be (8, 16, 24)
  • In order to Select no. of pairs, the Size Set field should be selected. If Size set is not selected, and user clicks on No. of Pairs dropdown, show toast "Please select a Size Set first"
2Fulfill pairs box

Suppose, value of MOQ/Carton = X Pairs,

By Default, when no. of Pairs value is not selected in any variant block, keep this box hidden. 

Now if the sum of pairs selected in all variant block is less than 'X', show this box with following text:

  • Text: "Required minimum of X Pairs"

As long as the minimum pairs criteria is not fulfilled, the "Add to Truck" button must be in Disabled State.

As the sum of pairs selected in all variant block is equal or more than 'X', hide this box again.
