Documentation : What We Offer

Created by Siddharth Rastogi, last modified on Mar 25, 2018

Overview

This is basically a web page that will be accessible from two places.

a. "What we offer" slider container on home page

b. "What we offer" menu option in the side drawer.

The objective of this is to clearly display the services that ShoeKonnect has to offer. 

Structure and Content

This page will show services as list of a collapsible/expandable rows.

If you try to access this page via a specific service block on Home Page Container,  this web page will open and the particular service block will be in expanded state.

Whereas if you try to access this page via Side Drawer menu, the web page will open up with all the services in collapsed state.

Contents:

  • Icon
  • Title
  • Main Content
  • Customer Care Call Button
