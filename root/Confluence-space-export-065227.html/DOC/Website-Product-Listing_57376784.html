Documentation : Website Product Listing

Created by Tushar Gupta, last modified on Apr 27, 2018

Overview

Product listing website offers buyers to view all type of products associated with the brand. Products can be filtered based on the desired choices. 
Sellers can share their branding pages on various social platforms and get recognised among the targeted audience.

Data Points & Logic (Brand Header)

#Data Points & ButtonsLogic
1Brand LogoShow the logo of the brand.
2Brand Name Mention the name of the brand. <In case of the full width >
3Seller NameMention the name which is registered. 
4Total ProductsTotal number of products loaded by seller in a particular brand
5LocationMention the city name
6Bread crumsCharacter limit = 20; post limit display with two dots '..'
7Share (FB/ Twitter/ LinkedIn)Using Social media API. Allow user to post the link with the OG image which is editable.
