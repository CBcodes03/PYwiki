Documentation : Favourites

Created by Siddharth Rastogi, last modified on Mar 24, 2018

Overview

User can mark products as favorite. They can go to "My favorites" and revisit all the products marked as favorites.

User Flow

  1. User can click favorite icon on the product card or the product list item
  2. User can slide the product list item in one direction to mark it as favorite (We can do this in subsequent release)
  3. "My Favorites" menu item will be there in Side Navigation menu which will display all the favorites marked items of that user.

Validations

When user is trying to mark a product as favorite, check whether logged-in or not. If Not, show toast "You need to login to use this feature" and open the Login/Signup Page.

Same goes when a user trying to access "My Favorites" menu without logging in.
