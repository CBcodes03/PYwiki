Documentation : Order Details > Summary

Created by Siddharth Rastogi, last modified on Apr 16, 2018

Overview

This is the second tab of the 'Order Details Page'. This page is broadly divided into X Sections:

  1. Header
  2. Current Status
  3. Original Bill Details
  4. Advance Payment (optional)
  5. Delivery Address


Data Points and Logic - Header

#
Data Points / Buttons
Logic
1Seller NameCompany Name of the Seller
2Seller MOQValue of Seller MOQ (this must be fetched from the orders table, as this will be save in orders table at the time of placing the order.)
3Order IDSK Order ID generated
4Order ValueThis is the total net order value (adjusted of shipping and discounts)
5Total Pairs Total sum of pairs in the order
6Payment OptionPayment option of the order

Data Points and Logic - Current Status

This data will be calculated at run time.

#
Data Points / Buttons
Logic
1CancelledTotal Pairs & Amount cancelled in that Order
2RTOTotal Pairs and Amount of pairs RTO in that Order
3DeliveredTotal Pairs and Amount of Pairs Delivered in that Order
4UndeliveredTotal Pairs and Amount of Pairs which are not Delivered, RTO or Cancelled

Data Points and Logic - Original Bill Details

#
Data Points / Buttons
Logic
1PairsTotal Pairs ordered
2SubtotalSum Value of all the pairs ordered (SKPrice + extSKPrice)
3Less: Coupon DiscountThis value will be deducted from subtotal
4Add: Shipping ChargeThis value will be added to the subtotal
5Less: Advance DiscountThis value will be deducted from Subtotal. Do not show this row if value is 0
6Grand TotalThis is the sum value of all the Value given above.

Data Points and Logic - Advance Payment

This box will appear only in case the payment Option is 20% Advance.

#
Data Points / Buttons
Logic
120% Advance 20% of the Net Amount
280% COD80% of the Net Amount
3Advance Payment20% of the Net Amount

Delivery Address

This will show the delivery address given by the Buyer

  1. Company Name (Bold)
  2. Representative Name
  3. Phone No
  4. Address Line 1 
  5. City - PinCode
  6. State
