Documentation : Case 3: Carton Fixed (Include Color = Yes)

Created by Siddharth Rastogi on Mar 24, 2018

This case has some Ui and Logic changes as mentioned below:

Data Points & Logic

S.No
Data Points 
Logic
1Variant Block

In this particular case, by default show all the colors configured by the Seller in different variant blocks. 

  • Populate these blocks with the corresponding Size Set and No. of pairs value.
  • Both no. of Pairs and Size Sets dropdowns must remain in a locked state (cannot be changed)
  • Hide the 'Change Color' button and 'remove' button.
  • Show the SK price and Pairs value and the Total Amount as pre-filled. 
2Choose No. of Cartons

The dropdown value will give the list of numbers starting from 1 upto 20.

  • Validate for Stock Lot information. If Stock lot is activated and the value of carton stock count is say 3, then in the dropdown, the values populated should be (1,2,3)
  • By default, the value in this dropdown should be set to 1.
  • 1 Carton = Value of MOQ Pairs, therefore if user selects no. of cartons as 2 and MOQ Pairs = 24, then at the bottom show total pairs as (2 x 24 = 48 Pairs)
