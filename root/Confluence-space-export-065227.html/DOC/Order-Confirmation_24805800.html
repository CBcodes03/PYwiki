Documentation : Order Confirmation

Created by Siddharth Rastogi on Mar 25, 2018

Overview

This screen appears after an order is successfully placed. This page will show an overview of the order placed and some other details.

There are two variations in the screen:

a. 20% Advance

b. Verification warning

Data Points and Logic

S.NoData Points / ButtonsLogic
1Total Order AmountShow the Total Order Amount value
2On SellersShow the seller names and their respective order ids created.
320% Advance Box

This box will appear when payment option selected is 20% Advance. 

Show text  of breakup and "Received Payment" and its value on successful payment.

4Verification Warning

Show this message if the user has not given any verification document. This means user has not provided any of the following:

  1. GST no. 
  2. Document type and Document URL

The warning text must be dynamic. Text: "Please provide any verification document or GST No. to place your order or else your order will be cancelled."

There will be a button "Click here to give verification". On click, this button, open the GST Verification Details page. 

5Go Home ButtonOn click, open the Home page. Do the same on click of physical back button.
6Call Us ButtonOn click, open Dialier
7My Orders ButtonOn click, open My Order > All Orders Page
