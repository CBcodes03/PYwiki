Documentation : Side Drawer Menu

Created by Siddharth Rastogi, last modified on Mar 31, 2018

Menu items:

  1. My Orders
  2. Track Consignments
  3. Returns and Refunds
  4. My Favorites
  5. GST and Bank Details
  6. What We Offer
  7. Sell on ShoeKonnect
  8. Call us (Customer Support)
  9. Settings
  10. Sign Out (Hidden in case user is logged out)
  11. App Version (not clickable)

Profile image:

  1. Profile Image - If logged in and provided image
  2. Company Name initial - if logged in and not provided image
  3. Generic User alert icon - if not logged in

Profile Details (If Logged In)

  1. Company Name
  2. Profile Type

Buttons 

  1. If Logged in, show one button "Edit Profile". On its click, open the edit profile page.

  2. If not Logged in, show two button "Register" & "Log In", on click of which open the subsequent tabs of Login/signup page. (To be decided whether needs redesign or existing design to be used) 

Validations

While user is not logged in and tries to click on the following menu items:

a. My Orders

b. Track Consignments

c. Returns & Refunds

d. My Favourites

e. GST & Bank Details

Show toast "Please register or login to use this feature" and open the Login/Signup Page
