7Variant Color

In case product has only one variant color, then hide the colors area and reduce the height of the image Image Slider container accordingly.

In case product has more than one variant color, then show the color area with following elements

  • Outlined box with variant thumbnail
  • Rectangle filled with the color of the variant
  • In Selected state, the outline/border of the box will become green in color with increased stroke width.
  • Also, in case there are many variant colors, this area should become horizontally scrollable to accommodate all colors.
  • On click of any color, refresh the page and update only the slider images as rest of the things will remain same.

11Previous/Next Product Button

On click of these buttons a different product of the same seller should open. We have to have a sliding transition animation between changing of products. 

We can use the facebook shimmer effect while the data loads on the new product page.

The logic of previous/next variants will be Randomized initially and we will change this logic eventually. Therefore, this logic must be handled dynamically via APIs.

While fetching products run them through the availability logic.
