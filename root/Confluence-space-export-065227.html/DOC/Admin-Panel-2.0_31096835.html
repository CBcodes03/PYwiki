Documentation : Admin Panel 2.0

Created by Siddharth Rastogi, last modified on May 17, 2018

Overview

This web panel will accommodate all the operational requirements of the internal team ranging from Verifying orders to providing refunds on cancellation. It will function on a Roles and Responsibilities framework where team's user profiles will be created and assigned roles and responsibilities on the basis of which various permissions to access or edit various modules in the admin panel will be granted.
