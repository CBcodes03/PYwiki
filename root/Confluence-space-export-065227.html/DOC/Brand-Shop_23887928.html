Documentation : Brand Shop

Created by Siddharth Rastogi, last modified on Mar 26, 2018

Overview

After the introduction of Brand Management System, one Brand could be sold by Multiple sellers. Therefore, any single brand can have its products sold by various sellers and therefore we need to address this as a seller filter feature in brand shop.

Data Points

S.NoData Points / ButtonsLogic
1Brand NameName of the Brand opened
2Brand Logo 

If the brand logo is present, then show logo.

Else, show the initial of the brand name. 

3Total Products CountShow count total no. of available variants in that brand. 
4Seller Filter Feature

Case 1 - If the brand has only one Seller, then this filter must remain hidden.

Case 2 - If there are more than one sellers of the brand, then this filter will be visible.

Show "count of Sellers" and a dropdown dialog, on click of which a popup will open showing the list of sellers selling this brand. (Popup Dialog explained below in further detail)

If there are total 3 sellers and user deselects one seller, then the text here should be "Showing 2 out of 3 Sellers"

5Product ListingBasic Product Listing
6FiltersWe will use existing filters. (New Filters will be introduced in subsequent release)

Note* - By Default, all the sellers will be enabled.



    

Seller Filter Popup

Data Points

S.NoData Points / ButtonLogic
1Showing SellersShow the count of Sellers showing
2Checkbox List

List all the Sellers of the Brand.

By default, all sellers must be checked.

3Seller Name

Company Name of the Seller

4Brand Owner tag

Show this tag against the seller name if that seller is the original seller of the brand.

There can be only one Brand Owner for one brand.

5Total ProductsBelow every seller, show the total count of variants that seller has for that brand. 
6Apply ButtonValidation: If no seller is checked and user clicks on 'Apply' button, then show toast "Please select any one seller"

Note - If user clicks outside the popup, it should close.
