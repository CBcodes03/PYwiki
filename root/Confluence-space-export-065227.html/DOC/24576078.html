Data Points & Logic

Data Points 
Logic
1No. of Pairs

The dropdown value will give the list of numbers in multiple of Lot Size. 

Minimum being the Lot Size value itself and going upto 10 multiples (dynamic).

  • Validate for Stock Lot information. If Stock lot is activated and the value for a particular size set is say 3 Lots ; where lot size = 8, then in the dropdown, the values populated should be (8, 16, 24). 
  • In order to Select no. of pairs, the Size Set field should be selected. If Size set is not selected, and user clicks on No. of Pairs dropdown, show toast "Please select a Size Set first"
2Fulfill pairs box

Suppose, value of MOQ/Carton = X Pairs,

By Default, when no. of Pairs value is not selected in any variant block, keep this box hidden. 

Now if the sum of pairs selected in all variant block is not equal to 'X' or any multiple of 'X', show this box with following text:

  • If sum of no. of pairs selected is less than X, show text: "Required minimum of X pairs"
  • If sum of no. of pairs selected is more than X, check for nearest lower and upper multiples of X to that value. and show Text "Please choose either X1 pairs or X2 pairs" ; where X1 = lower nearest multiple AND X2 = upper nearest multiple. 

As long as the required pairs criteria is not fulfilled, the "Add to Truck" button must be in Disabled State.

As soon as the required pairs criteria is met, hide this box again.
