Documentation : Seller Listing

Created by Siddharth Rastogi, last modified on Mar 26, 2018

Overview

This will be a new page introduced to the app. this will help users browse through the sellers present on the App. Only the available sellers must appear in any seller listing (logic for sellers defined below).

Data Points

S.NoData PointsLogic
1Seller NameName of the Company with which registered as seller.
2CityName of the city with which registered
3Seller MOQ

If the value is more than 0, show "Seller MOQ - X Pairs"

whereas, if the value is "0", show "No Seller MOQ"

4Product CountShow Total active products of that seller. (logic of availability applicable here)
5Image Thumbnails

Show 4 variant thumbnail images. the variants visible here will be according to the latest product approved.Â 

6Go to Seller Shop (button)redirect to Seller Shop Info Page

User flow

  1. If user click on any area of the card except the image thumbnails, then "Seller Shop Info" page will open up.
  2. If user click directly on any image thumbnail, then "Product Details" page will open up.
  3. There will be no Seller filters for release 6.0


Logic to check availability

  1. sellerStatus = Active
  2. isProductActive = 1
  3. Has minimum of one approved product with status "isAvailable=true" and "brand status = Active" and limited stock for all size sets is not "0" (if limited stock is active)
