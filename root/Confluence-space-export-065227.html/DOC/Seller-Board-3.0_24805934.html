Documentation : Seller Board 3.0

Created by Siddharth Rastogi, last modified on Mar 27, 2018

Overview

This Release is a step to increase our products adaptability with various types of sellers present in the Footwear Industry. This release will give more flexibility and control to the sellers and position our brand as a pure marketplace.:

Changes to be be made for following concepts:

  • Product MOQ
  • Seller MOQ
  • Size Sets at Product Level (instead variant Level)
  • Various Sell Types
  • Cancellations
