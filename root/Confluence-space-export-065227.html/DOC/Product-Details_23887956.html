Documentation : Product Details

Created by Siddharth Rastogi, last modified on Jun 16, 2018

Overview

This is a revamp of the existing product details page. Goal has been to give the buyers a feel of looking at products inside a seller shop. Also, enabling buyers to explore more products through this page and avoid a dead-end.

In case product has only one variant color, then hide the colors area and reduce the height of the image Image Slider container accordingly.

Updated Logic for info text

Case 1:

Sell Type = Lot, Acc: Yes, (1+ Variants)

  1. You can buy minimum of X Pairs of multiple color. ;Where X = MOQ Pairs

Case 2:

Sell Type = Lot, Acc: Yes (1 Variants) OR Acc: No 

  1. You can buy minimum of X Pairs of single color. ;Where X = MOQ Pairs

Case 3

[ Sell Type = Carton Fixed, Acc: Yes (1+ Variants) ] 

  1. Carton Type: Multiple Colors per Carton (Fixed)

Case 4

[ Sell Type = Carton Fixed, Acc: No ]  OR  [ Sell Type = Carton Fixed, Acc: Yes (1 Variants) ] OR [ Sell Type = Carton Flexible (1 Variants) ]

  1. Single Color per Carton

Case 5

Sell Type = Carton Flexible (1+ Variants) 

  1. Carton Type: Multiple Colors per Carton (Customizable)





OLD Logic - Do not Look Here 

Logic for Buying Details Text


Case 1: Sell Type = Lot, Include Color = No,

"You can buy minimum of X pairs of single color" ; where X = MOQ Pairs

Case 2: Sell Type = Lot, Include Color = Yes, (more than one variant)

"You can buy minimum of X pairs including multiple colors" ; where X = MOQ Pairs 

Case 3: Sell Type = Lot, Include Color = Yes, (one variant)

"You can buy minimum of X pairs of single color" ; where X = MOQ Pairs 

Case 4: Sell Type = Carton, Carton type = Fixed, Include Color = Yes

"You can buy minimum of X pairs of fixed colors and Size set" ; where X = MOQ Pairs 

Case 5: Sell Type = Carton, Carton type = Fixed, Include Color = No

"You can buy minimum of X pairs of single color" ; where X = MOQ Pairs 

Case 6: Sell Type = Carton, Carton type = Flexible (one variant)

"You can buy minimum of X pairs of single color"; where X = MOQ Pairs

Case 7: Sell Type = Carton, Carton type = Flexible (more than one variant)

"You can buy minimum of X pairs including multiple colors" ; where X = MOQ Pairs

Case 8: Sell Type = Lot, and MOQ = Lot Size

"You can buy minimum of X Pairs of single color"; where X = MOQ Pairs
