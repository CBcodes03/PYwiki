Documentation : Buyer Info PopUp

Created by Siddharth Rastogi, last modified on Apr 14, 2018

Overview

This popup can be accessed by clicking on the info icon next to the Buyer Name (City) in "New Orders" or "In Process" Screen.

This popup will basically give more details about the order and also some buyer history of the seller.

Data Points & Logic

S.NoData PointsLogic
1Buyer Name This will be the title of the popup
2Profile TypeThis will be the subtitle of the popup
3Order Count

Count of Orders that buyer has placed on the Seller before 

Keep this line hidden if the count is 0.

4Pie Chart

Represent the following data in a pie chart.

  1. Cancellation Rate – Total Pairs Cancelled by Buyer divided by the Total Pairs Ordered.
  2. RTO rate - Total Pairs RTO divided by Total pairs ordered.

This Pie Chart will be visible only once the Buyer has passed a threshold Parameter (Updated below the image). 

In case the threshold has not been passed, keep the pie chart hidden and shift the "order details" section towards the left. (The width of popup must reduce accordingly.) 

5Order Details

This is a tabular format element. Show following contents:

  1. Total Amount: Total Order Value (NET)
  2. Total pairs
  3. Payment Option
  4. GST No. (If any)
  5. Full Address
  6. City
  7. Pincode
  8. State


Logic for Showing Pie Chart Data

Show the pie chart data if:

  1. The total order count of the buyer is 6. (i.e o.status=4 AND o.confirmedOn>0)
  2. The Cancellation Rate Value and RTO Rater Value should be greater than 0%
